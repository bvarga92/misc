# Peremértékfeladatok megoldása PDE Toolbox segítségével

- **heat**: egyszerû kétdimenziós hõvezetési probléma végeselemes szimulációja
- **membrane**: széleinél rögzített, kör alakú membrán rezgésének végeselemes szimulációja
- **waveguide**: végeselemes szimuláció az elektromágneses hullám csõtápvonalban való terjedésére  ([videó](https://www.youtube.com/watch?v=mLOpLBptPcE&list=PL9_VlVdB8s882QMHiqJlDpJeKWxwP5CIG))
- **coax_parameters.m**: koaxiális kábel EM terének szimulációja és hullámimpedanciájának számítása
- **pcb_eigenmodes.m**: nyomtatott áramköri hordozón kialakuló rezgésmódusok meghatározása
